# Image Prompt Engineering for Gemini

## Base Template
`"{Subject}, {style}, {lighting}, {composition}, 9:16 vertical portrait orientation, high quality"`

## Required Elements
1. **Subject**: Clear, specific description of the main visual
2. **Style**: "photorealistic", "digital illustration", "cinematic", "anime style"
3. **Lighting**: "warm golden hour", "neon cyberpunk", "soft studio lighting"
4. **Composition**: "9:16 vertical", "centered subject", "rule of thirds"
5. **Quality**: "high quality", "detailed", "4K", "professional"

## Do NOT Include
- Text or words in the image (renders poorly)
- Human faces in close-up (inconsistent quality)
- Copyrighted characters or logos

## Style Consistency
- Maintain consistent style across ALL scenes in a video
- Use similar color palettes throughout
- If scene 1 is "cinematic digital art", all scenes should match

## Effective Patterns
- Contrasts: "dark background with bright focal point"
- Depth: "foreground elements with blurred background"
- Emotion: warm colors for positive, cool for serious
- Abstract concepts: represent as visual metaphors

## Aspect Ratio
ALWAYS specify "9:16 vertical portrait orientation" or "vertical composition".
Target: 1080x1920 pixels.
